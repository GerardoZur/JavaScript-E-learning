Unidad 2: Variables y Operadores

Tipos de variables - Arrays o Vectores
Operadores - Cuadros de diálogo

El segundo práctico consistió en armar un código para generar recetas de ensaladas personalizadas, en este caso de 3 ingredientes.
Mediante variables que almacenan los datos que ingresa el usuario a traves de prompt(), preguntará:
-Nombre de la ensalada
-Nombre de cada uno de los ingredientes
-Proporción de los ingredientes
Finalmente, una variable llamada "resultado" verificará que las proporciones indicadas sumen 100%. En caso contrario advertirá al usuario, ya que la informacion podría mal interpretarse.
La impresión de las variables se realiza en el archivo HTML utilizando document.write().

Unidad 3: Vectores y Ciclos

Vectores simples y complejos - Métodos
Bucles: For, While, Do While

El trabajo práctico tenía 2 consignas:
-La primera consistió en generar 2 variables numéricas, operando A y operando B, que luego fueron sometidas a la suma, resta, multiplicación y división, poniendo en práctica el uso de operadores, pendiente de la unidad anterior
-La segunda consigna nos pedía generar un array de 5 elementos, estos serían nuestras 5 películas favoritas.
Tanto las variables numéricas como el resultado de las operaciones se imprimieron en pantalla con el método document.write().
La impresión de las 5 películas se hizo mediante el mismo método pero dentro de una estructura de control, un bucle For para iterar el contenido del Array. La condición establecida en el bucle se hizo con el método .lenght estableciendo el máximo de 5 repeticiones de la estructura (longitud del array) hasta finalizar.
